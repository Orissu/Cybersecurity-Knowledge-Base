## 🚀 General Methodology (Kill Chain)
1. **Reconnaissance**: Passive & Active discovery.
2. **Weaponization**: Preparing payloads and exploits.
3. **Delivery**: Sending the payload to the target (Phishing, Web exploit).
4. **Exploitation**: Triggering the vulnerability.
5. **Installation/Persistence**: Maintaining access.
6. **Command & Control (C2)**: Remote management of the target.
7. **Actions on Objectives**: Exfiltration or data encryption.
