# 🧰 Security Toolkit & Cheat Sheets

> **Objective:** Atomic documentation for specific security tools. Each file contains syntax, flags, and expert one-liners.

## ⚔️ Offensive Tools (Red Team)
* **[Nmap](./nmap.md)**: Network discovery and security auditing.
* **[Sqlmap](./sqlmap.md)**: Automated SQL injection and database takeover.
* **[Gobuster](./gobuster.md)**: URI and DNS brute-forcing.
* **[Metasploit](./metasploit.md)**: Exploitation and post-exploitation framework.

## 🛡️ Defensive Tools (Blue Team)
* **[Wireshark](./wireshark.md)**: Network protocol analyzer.
* **[Snort](./snort.md)**: Open-source Intrusion Detection System.
* **[Splunk](./splunk.md)**: SIEM and data analytics platform.
* **[Tcpdump](./tcpdump.md)**: Command-line packet analyzer.
